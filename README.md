# Agenda de Contatos

Projeto prático de versionamento e automação contínua para uma aplicação de Agenda de Contatos.

## Funcionalidades Principais

- **Cadastro e Listagem de Contatos** - `feature/contatos`
- **Marcação e Desmarcação de Contatos Favoritos** - `feature/favoritos`

## Tecnologias

- Git/GitHub para versionamento
- SemVer para versionamento semântico

## Como Usar

Este é um projeto em desenvolvimento. Para começar, consulte as branches de features para mais informações.
